Note

This list’s formatting is “Applications name” and “short description”. Applications name may vary among disto to distro and may have multiple sources along with documentations. This is why I’ll recommend you to use a search engine.

This is my personal list that I’ve tried. If you need more detailed and vast list, try the following links:

  • Awesome-Linux-Software from “Luong Vo”
